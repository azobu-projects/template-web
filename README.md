## Getting Startedf

### Installation

Install dependencies using `yarn`:

```sh
yarn
```

You should also install and run `template-api` before continuing.

### Run

```sh
yarn dev
```

Access it on <http://localhost:8000>.

### Lint

```sh
yarn lint
```

To fix basic issues:

```sh
yarn lint --fix
```
